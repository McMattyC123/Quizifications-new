# App Store Submission Files

This folder contains all metadata needed for iOS App Store and Google Play Store submissions.

## Files

| File | Purpose |
|------|---------|
| `ios-app-store-metadata.json` | Complete iOS App Store Connect metadata |
| `google-play-metadata.json` | Complete Google Play Console metadata |
| `app-store-description.txt` | Plain text descriptions for copy/paste |

## Quick Reference

- **App Name:** Quizifications
- **Category:** Education
- **Price:** Free (with in-app subscription)
- **Trial:** 3 days free
- **Monthly:** $4.99
- **Yearly:** $39.99 (save 33%)

## Required Assets (create separately)

### iOS App Store
- App Icon: 1024x1024 PNG
- Screenshots: 6.5" (1284x2778), 5.5" (1242x2208)

### Google Play
- App Icon: 512x512 PNG  
- Feature Graphic: 1024x500 PNG
- Screenshots: min 2, recommended 8

## Submission Checklist

### iOS App Store Connect
- [ ] Upload app icon (1024x1024)
- [ ] Add screenshots for required device sizes
- [ ] Copy app name, subtitle, description from metadata
- [ ] Add keywords
- [ ] Set category to Education
- [ ] Add privacy policy URL
- [ ] Configure in-app purchases
- [ ] Set age rating (4+)
- [ ] Submit for review

### Google Play Console
- [ ] Upload app icon (512x512)
- [ ] Upload feature graphic (1024x500)
- [ ] Add screenshots
- [ ] Copy short and full description
- [ ] Complete content rating questionnaire
- [ ] Complete data safety form
- [ ] Add privacy policy URL
- [ ] Configure subscriptions
- [ ] Submit for review
